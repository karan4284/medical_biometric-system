# Medical Biometric System – MedLife

## Project Purpose
This project aims to design a medical equipment–integrated biometric system
to capture and store biometric and basic medical data of newly hired employees
at MedLife during the onboarding process.

The system ensures accuracy, security, and efficiency in employee health
verification and onboarding approval.

---

## Project Objectives
- Capture biometric and medical parameters of new employees
- Securely store employee health records
- Automate employee medical verification
- Support HR and medical approval workflows

---

## System Features
- Employee registration
- Biometric data capture
- Medical parameter validation
- Secure database storage
- HR and medical approval
- Report generation

---

## Technologies Used
- Programming Language: Python
- Database: MySQL
- Tools: GitHub, Draw.io
- Hardware: Biometric & Medical Sensors
